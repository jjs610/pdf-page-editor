# PDF Page Editor

브라우저에서 바로 실행하는 로컬 PDF 페이지 편집기입니다. 파일은 사용자의 브라우저 안에서 처리되며 서버로 업로드하지 않습니다.

## 사용 방법

1. `index.html` 파일을 브라우저로 엽니다.
2. `PDF 열기`로 편집할 파일을 선택합니다.
3. 기본 화면에서는 각 페이지가 낱장 썸네일로 크게 표시됩니다.
4. `파일 추가`는 다른 PDF를 맨 뒤에 붙입니다. PDF 파일을 이 버튼 위로 드래그 앤 드롭해도 됩니다.
5. `삽입`은 선택한 페이지 바로 뒤에 다른 PDF를 끼워 넣습니다. PDF 파일을 이 버튼 위로 드래그 앤 드롭해도 됩니다.
6. 페이지를 더블클릭하면 큰 미리보기가 열리고, `닫기`를 누르면 다시 낱장 썸네일 화면으로 돌아갑니다.
7. `Ctrl`을 누른 채 페이지를 클릭하면 현재 활성 페이지와 클릭한 페이지가 함께 개별 선택됩니다. 이후 `Ctrl` 클릭은 해당 페이지만 추가/해제합니다.
8. `Shift`를 누른 채 페이지를 클릭하면 기준 페이지부터 클릭한 페이지까지 범위로 선택됩니다.
9. 선택된 페이지가 여러 개 있을 때 `Ctrl` 없이 다른 페이지를 클릭하면 선택이 해제되고 그 페이지만 활성화됩니다.
10. 썸네일 목록의 빈 공간에서 드래그하면 현재 활성 페이지와 영역에 걸친 페이지가 함께 선택됩니다.
11. 선택된 페이지가 있는 상태에서 `저장`을 누르면 선택한 페이지만 저장합니다.
12. 선택된 페이지가 없을 때 `저장`을 누르면 편집된 전체 PDF를 저장합니다. 지원 브라우저에서는 파일명과 저장 위치를 직접 선택할 수 있고, 다음 저장 시 마지막 저장 위치에서 시작합니다.
13. 낱장 페이지를 드래그해서 순서를 바꿀 수 있습니다.
14. `Delete` 키를 누르면 `Ctrl`로 선택한 페이지를 삭제합니다. 선택한 페이지가 없으면 현재 선택된 페이지를 삭제합니다.
15. PDF 파일을 화면에 드래그 앤 드롭하면 기존 문서를 대체해서 새로 엽니다.

## 지원 기능

- PDF 미리보기
- 페이지 선택
- 더블클릭 큰 미리보기
- 페이지 드래그 순서 변경
- 선택 페이지 90도 회전
- 파일 추가로 여러 PDF 병합
- 삽입으로 선택 페이지 뒤에 PDF 파일 추가
- `Ctrl` 클릭 개별 선택
- `Shift` 클릭 범위 선택
- 드래그 영역 선택
- 선택한 페이지만 저장
- `Delete` 키로 페이지 삭제
- 저장할 때 파일명과 위치 선택 및 마지막 저장 위치 기억
- 드래그 앤 드롭으로 새 PDF 열기
- 편집된 PDF 다운로드

## 참고

이 버전은 페이지 단위 편집기입니다. 기존 PDF 안의 문장 자체를 직접 수정하는 기능은 포함하지 않았습니다.
